lambda(s)

Compiles a string representing a lambda expression into a Lua function.

A lambda expression has this form:

(input-parameters) => expression

Parameters:

  • s string a valid lambda string.

Returns:

  1. function the compiled lambda string, or nil if the compilation fails.
  2. string an error message if the compilation fails, otherwise nil.
